## main.py

요기요 리뷰 api url이 식당코드와, pageNum을 인자로 받아 데이터를 전송해줌.
반복문을 활용하여 데이터 수집.
별점과 같은 데이터를 함께 사용하기 위해 sql을 사용하였음.

혹시모를 서버 과부화 방지위해 sleep 적용
